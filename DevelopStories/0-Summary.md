### BasePrinciples

| 尽量可复用       | Reusable as much as possible                         |
| ---------------- | ---------------------------------------------------- |
| **尽量逻辑抽象** | **Try to be logically abstract as much as possible** |
| **尽量简单易懂** | **Try to be simple and easy to understand**          |
